# anki-biaori

生成《新标准日本语》初级基本课文 Anki 卡组。

卡片正面为中文翻译，背面为日文课文并带日语发音。

### Code

使用 [furigana](https://github.com/MikimotoH/furigana) 生成注音假名，并手动更改了标注错误的地方（[64efd3b](64efd3b)）。如发现仍有错误，欢迎 issue / PR。
